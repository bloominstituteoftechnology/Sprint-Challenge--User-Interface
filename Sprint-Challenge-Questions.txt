2. Name two big differences between display: block; and display: inline;.
    
     Two big differences between display:block; and display: inline are that display block elements can only be nested within new blocks level elements, while inline elements can be nested with both block and inline elements.
     Another difference between the two are that display:block elements are can be subject to width and height properties whereas inline elements do not recognize neither weight or height.
